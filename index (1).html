How the import reads your Excel columns

The app looks for these column names in your Excel file. Your columns don't need to match exactly — it finds the closest match automatically.

College / Organisation Name
Any column with "college", "name", "organisation"
Visit Date
Any column with "date"
Contact Person
Any column with "contact", "person", "name"
Phone Number
Any column with "phone", "number", "mobile"
Outcome / Discussion
Any column with "outcome", "discussion", "details"
Next Action
Any column with "next", "action", "follow"
